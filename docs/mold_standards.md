# Mold Standards (v0)

Molds are planned but not validated yet. This file records:
- what “mold-ready” means for this project
- decisions and tests as molds begin

## When molds happen
- After a fully assembled master is finished to acceptable surface quality
- After segmentation + alignment approach is stable

## Mold-ready (definition of done)
- seams filled and not visible under primer
- surface smooth enough that mold will not capture print lines
- undercuts understood / managed (especially intakes)

## Open questions
- mold material choice (silicone, fiberglass, etc.)
- release system (wax/PVA)
- parting line strategy for YF-23 geometry
