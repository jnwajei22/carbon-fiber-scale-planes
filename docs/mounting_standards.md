# Mounting Standards (v0)

This project aims for wall-mountable display models. No mounting solution is finalized yet.

## Requirements
- mount must support model weight with safety margin
- mount must not crack the shell over time
- mount should be serviceable (removable)

## Approach (initial)
- Identify internal hardpoints or reinforcement areas
- Avoid relying on thin shell alone for load
- Prefer mounting that spreads load (plates/blocks) rather than point loads

## Safety rule (non-negotiable)
- First mounting test happens low to the ground
- Hardware ratings documented (anchors, wire, screws)
